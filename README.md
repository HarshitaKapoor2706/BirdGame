🐦 Bird Game — Android (Java)
A Flappy Bird-style Android game where you fly through enemies, collect coins, and survive with 3 lives to reach a score of 200!

🎮 Gameplay

Tap the screen to make the bird fly
Avoid enemies — touching one costs you a life
Collect coins to increase your score
Reach a score of 200 to win the game
You have 3 lives — lose all three and it's game over


✨ Features

🪙 Coin Collection — fly through coins to rack up your score
💀 Enemy Obstacles — dodge enemies or lose a life
❤️ 3 Lives System — you get three chances before game over
🏆 Win Condition — reach 200 points to win
📊 Score Tracking — live score displayed during gameplay
🔊 Sound Effects — audio feedback for coins, hits, and game events


🛠️ Built With

Language: Java
Platform: Android
IDE: Android Studio


🚀 How to Run

Clone the repository

git clone https://github.com/HarshitaKapoor2706/BirdGame.git

Open the project in Android Studio
Wait for Gradle sync to complete
Click Run ▶ to launch on an emulator or physical device
